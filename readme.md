# Wagon Fill Calculator

An online wagon utilisation calculator based on stillage-equivalent floor space.

## Rules
- 14 doors per stillage
- Only large pallets (2.8m)
- 1 stillage = 2.25 large pallets

## How to use
1. Select a vehicle
2. Enter door quantity and/or large pallet quantity
3. Review utilisation by floor space, cube, and weight

Built with Streamlit.
